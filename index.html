Web Designer Hanako Kirameki

Webデザイナーを目指している、きらめき華子です。現在、在籍している会社のイベント企画においてWeb制作に携わりはじめたことを契機に、より専門性を身に付けるためにオンラインスクールで学んでいます。これからさらにWeb制作の技術や知識を身につけて、既存サイトのリニューアルや、新規サイトの立ち上げ、さらにはWebサイト運用や問題解決のアドバイスもできるデザイナーになりたいと思っています。

Skills

Web

HTML/CSSでのコーディングができます。JavaScript、PHP、Ruby、WordPressも現在勉強中です。

analysis

イベントの企画運営経験を基にしたデータ分析を得意としています。集客のための訴求力を高める施作や満足度を上げるノウハウをwebにおいても活用してまいります。

Music

幼少時よりピアノを習い、現在はジャズピアノやロックバンドでの演奏を行っています。音楽活動の経験を活かし、演者やイベント主催者のニーズに沿ったデザインを心掛けています。
